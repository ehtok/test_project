# test_project
Есть It компании которые предоставляют:
- вакансии,
- курсы,
- стажировки.


У вакансии есть:
- описание вакансии
- режим работы
- локация
- специализация - опыт,название специализации,
зарплата, уровень английского, уровень 

У курсов есть:
- описание,
- время начало - конец
- локация

У стажировки есть:
- описние стажировки,
- локация,
- время начало - конец,
- навыки
- зарплата.
